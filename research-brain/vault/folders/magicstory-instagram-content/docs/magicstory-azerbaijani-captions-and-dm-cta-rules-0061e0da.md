# MagicStory Azerbaijani Captions and DM CTA Rules

## When to use this file

Use this when the user asks for a caption for a completed MagicStory post or wants short Azerbaijani copy for Instagram. The user preference is usually one final copyable caption, not a list of alternatives.

## Caption Goals

MagicStory captions should:

- be in Azerbaijani;
- be easy to copy;
- stay short enough for Instagram;
- feel emotional, warm, and parent-focused;
- focus on the child becoming the hero;
- mention personalized book or personalized story when relevant;
- present MagicStory as a meaningful gift and keepsake;
- include a simple DM call to action.

The caption should not over-explain the business unless the post itself needs clarification.

## Preferred Caption Structure

Use this three-part pattern:

```text
[Emotional hook about child / magic / story]

MagicStory ilə [child becomes hero / personalized story / emotional gift message] 💛

[CTA: Sifariş üçün DM yazın] 🌙📖
```

## CTA Rules

Keep the next step simple and low-pressure.

Preferred CTA lines:

```text
Sifariş üçün DM yazın
```

```text
DM yazın
```

Avoid complex CTAs. Instagram DM is an important sales channel, so captions should make users comfortable enough to message.

Good DM-oriented content should:

- build comfort;
- clarify the need;
- recommend specifically;
- reduce risk;
- ask for one easy next step.

For captions, this usually means a direct, friendly DM request rather than a long sales sequence.

## Example Caption: Heroic Horse / Adventure Post

```text
Hər uşağın içində bir qəhrəman yaşayır ✨

MagicStory ilə balacanız sadəcə kitab oxumur — öz sehrli dünyasında baş qəhrəmana çevrilir 💛

Adı, şəkli və sevdiyi macəralarla hazırlanan fərdi nağıl kitabı üçün bizə DM yazın 🌙📖
```

## Example Caption: Cap / Pin Mockup Post

```text
Balacanızın sehrli dünyası artıq bir simvola çevrilir ✨

MagicStory ilə uşağınız sadəcə kitab oxumur — öz adının, şəklinin və macərasının qəhrəmanı olur 💛

Onun üçün fərqli, şəxsi və ömürlük xatirə qalacaq hədiyyə axtarırsınızsa, bizə DM yazın 🌙📖

#MagicStory #SehrliHekaye #UsaqKitabi #FerdiKitab #UnikalHediyye #Azerbaycan #MagicStoryAz
```

Hashtags are optional. The user may accept hashtags, but when they ask simply for a caption, prioritize one clean caption over multiple versions.

## Birthday And Gift Framing

Birthday and special-occasion framing is important because many MagicStory orders are birthday-related.

Useful angles:

- meaningful birthday gift;
- personalized keepsake;
- order ahead because preparation takes time;
- child sees themselves as the hero;
- parent gives a memory, not only a book.

Do not introduce operational claims such as timing, price, or deposit unless the user specifically asks or provides that context for the post.

The 10 AZN deposit exists in broader business context, but it was not used in the caption examples from this source and should not be inserted automatically.

## Caption Quality Checklist

Before finalizing a caption, check:

- Is it in Azerbaijani?
- Is there exactly one caption if the user did not request variants?
- Does it mention the child as hero, personalized story, or emotional gift?
- Is it copyable without explanation around it?
- Is the CTA simple and DM-focused?
- Is it short enough to feel natural on Instagram?
- Does it avoid overcomplicated sales language?
- Does it avoid unsupported claims about price, delivery, or production timing?

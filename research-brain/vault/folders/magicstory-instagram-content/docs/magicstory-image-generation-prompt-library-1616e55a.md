# MagicStory Image Generation Prompt Library

## When to use this file

Use this when the user uploads a visual reference and asks to adapt it to MagicStory, or when an image-generation output needs correction for mascot identity, box accuracy, style fit, or brand consistency.

## Prompt Pattern 1: General Reference-Style Adaptation

Use when the user wants a MagicStory version of an uploaded reference.

```text
Create a new Instagram post image in the same overall visual style, mood, layout, and composition language as the uploaded reference image, but adapt it fully for the MagicStory brand.

Use MagicStory as a personalized children’s magical storybook brand. The child should feel like the hero of the story. Use warm golden-yellow, cream, beige, kraft brown, moon, stars, open book, and magical story elements. Do not copy the original brand, watermark, text, or exact characters from the reference. Use the reference only for visual inspiration.

Make the final image premium, child-friendly, magical, emotional, and Instagram-ready.

Aspect ratio: 3:4.
```

## Prompt Pattern 2: Mascot Reuse

Use when the MagicStory mascot should appear in a new style while staying recognizable.

```text
Use the provided MagicStory mascot reference as the character identity. Keep the same recognizable mascot: cute childlike magical boy, fluffy brown hair, big expressive eyes, warm friendly face, cream and golden-yellow outfit, small cape, and book/star details.

Transform the mascot into the visual style of the uploaded reference while keeping him clearly recognizable and aligned with MagicStory. Do not copy any third-party character. Make the result original, magical, premium, and child-friendly.

Aspect ratio: 3:4.
```

## Prompt Pattern 3: Packaging Integration Fix

Use when the MagicStory box looks pasted, generic, incorrectly shaped, or visually disconnected from the scene.

```text
Use the MagicStory packaging reference carefully. The box must match the real MagicStory kraft mailer gift box: flat rectangular kraft cardboard shape, correct folding edges, cream vertical wrap/sticker label across the lid/front, and MagicStory gold logo/text styling.

Integrate the box naturally into the scene. Match the scene’s lighting, shadows, perspective, scale, material texture, and art style. The box must look like it belongs inside the illustration/photo, not like it was pasted afterward.

If the character holds the box, make the hand grip, arm pose, gaze direction, and body orientation support the action naturally.
```

## Prompt Pattern 4: Child Information Collection Board

Use when the post should explain that MagicStory collects child details before creating the personalized book.

```text
Create a 3:4 social media poster in a scrapbook-like creative planning board style. The image should visually communicate that MagicStory first collects information about the child and then creates a personalized magical storybook.

Include a child profile card with placeholder details: name, age, favorite color, favorite animal, hobbies, favorite hero, and story theme. Add sticky notes or paper scraps labeled in Azerbaijani: “Ad”, “Yaş”, “Sevdiyi rəng”, “Sevdiyi heyvan”, “Maraqları”, “Sevdiyi qəhrəman”, “Mövzu seçimi”. Use arrows or visual flow from information gathering to finished book creation.

Show 2–3 MagicStory-style finished book mockups in a “Nəticə” or “Hazır kitab” area. Use warm golden-yellow, cream, beige, moon, stars, books, and MagicStory branding.

Main message: “Sizin verdiyiniz məlumatlarla onun öz hekayəsini yaradırıq”.

Aspect ratio: 3:4.
```

## Prompt Pattern 5: Heroic Fantasy Mascot Post

Use for adventure, fantasy, child-hero, mascot-led, or story-world posts.

```text
Create a heroic MagicStory fantasy adventure poster using the MagicStory mascot as the main character. The childlike mascot should look brave, joyful, and magical. Use a child-friendly heroic scene with books, stars, moon symbols, magical light, and storytelling motifs.

The scene should show that every child can become the hero of their own story. Keep the tone magical and adventurous but safe and suitable for children. Avoid violence or dark aggression. Use MagicStory colors: golden yellow, cream, warm beige, and deep magical blues/greens where needed.

No text unless requested. Aspect ratio: 3:4.
```

## Common Corrections

### If the box looks pasted

Add explicit instructions for:

- real packaging reference;
- kraft material;
- cream/gold vertical label placement;
- lighting match;
- perspective match;
- contact shadows;
- scale;
- hand grip if held;
- same art style as the scene.

### If the style does not match the reference

Restate that the prompt should preserve:

- visual style;
- mood;
- layout;
- composition language;
- texture level;
- graphic density;
- color relationship, adapted to MagicStory palette.

### If third-party similarity is too high

Do not name the protected property in the output prompt. Ask for:

- similar emotional composition;
- original childlike or fairy-tale characters;
- MagicStory brand world;
- no direct recreation of specific copyrighted characters or scenes.

### If generated text is weak

Use minimal image text or generate without text, then add typography manually later. Keep Azerbaijani phrases short.

## Default Output Constraints

Unless the user says otherwise:

- aspect ratio: `3:4`;
- platform: Instagram feed or carousel;
- language: Azerbaijani when text appears;
- visual tone: premium, magical, warm, emotional, child-friendly;
- core message: child becomes the hero of a personalized story;
- CTA if included: simple DM action.
